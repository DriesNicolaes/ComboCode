  44      The differential equation for the kinetic temperature profile.  
  45       
  46      Currently only takes into account adiabatic expansion.  
  47       
  48      This function is used by EnergyBalance to iterate and solve the ODE. 
  49       
  50      @param T: The temperature at which to evaluate the differential equation 
  51      @type T: float 
  52      @param r: The radial point(s) (cm) 
  53      @type r: array/float 
  54      @param v: The velocity profile object 
  55      @type v: Velocity()  
  56      @param gamma: The adiabatic coefficient profile as function of T 
  57      @type gamma: Profiler() 
  58       
  59      @keyword rates: The heating and cooling rates, summed up (erg/s/cm3, H-C).  
  60                      Includes the density factor, without the velocity component. 
  61                      This speeds up iteration, limiting the radial grid  
  62                      evaluation where possible. Default if only adiabatic cooling 
  63                      is taken into account. 
  64                 
  65                      (default: None) 
  66      @type rates: Profiler() 
  67      @keyword warn: Warn when extrapolation occurs in an interpolation object.  
  68                     Not applicable to functional evaluation, or gamma in which  
  69                     case the extrapolation is deemed safe (constant value at  
  70                     low and high end temperatures consistent with diatomic gases) 
  71       
  72                     (default: 1) 
  73      @type warn: bool     
  74       
  75      @return: The derivative with respect to radius (K/cm) 
  76      @rtype: array/float 
  77       
  78      ''' 
  79       
  80      #-- Set the gamma-based factor. gamma is assumed to be a Profiler object. 
  81      tg = gamma.eval(T,warn=0) 
  82      factor1 = 2-2*tg 
  83      factor2 = tg-1 
  84       
  85      #-- Determine the velocity and its derivative from the Velocity() object 
  86      vi = v.eval(r,warn=warn) 
  87      dvi = v.diff(r,warn=warn) 
  88       
  89      #-- Calculate dT/dr 
  90      #   1) adiabatic cooling term 
  91      term1 = factor1 * (1./r + 0.5*1./vi*dvi) * T 
  92       
  93      #   2) other heating and cooling terms (can be none at all). This includes 
  94      #      the density factor, except the velocity. 
  95      rr = rates.eval(r,warn=warn) if rates else 0. 
  96      term2 = factor2 * rr / vi 
  97       
  98      #-- And the final summation of the two terms 
  99      Tprime = term1 + term2
