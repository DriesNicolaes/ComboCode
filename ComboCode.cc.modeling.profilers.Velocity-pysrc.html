 24      ''' 
 25      Calculate the drift velocity as a function of radius and grain size by  
 26      equating the radiation pressure force and drag force.  
 27       
 28      If the grain size is given as a constant value, the profile is calculated 
 29      only for an average grain size, and is essentially 1d, in which case the 
 30      drift does not depend on grain size. 
 31       
 32      For now in the approximation that the thermal velocities are small compared 
 33      to the drift. This is only valid in the outer, cool regions of the wind.  
 34       
 35      The function integrates the luminosity and the opacity over the wavelength  
 36      grid to estimate the component for the radiation pressure. 
 37       
 38      Inclusion of the thermal velocity requires T profile. Mean molecular weight 
 39      assumed to be one in that case.  
 40       
 41      @param r: The radial points (cm) 
 42      @type r: array/float

 44      @type a: array/float 
 45      @param l: The wavelength grid (cm) 
 46      @type l: array/float

 48      @type v: float/Velocity() 
 49      @param mdot: The mass-loss profile (Msun/yr) 
 50      @type mdot: float/Mdot() 
 51      @param opac: The opacity profile (cm2/g), must include l-dependence. Can be 
 52                   extinction, absorption, or scattering, but note that the 
 53                   calculation assumes this is an absorption opacity. 
 54      @type opac: Opacity() 
 55      @param sd: The specific density of the dust species 
 56      @type sd: float 
 57      @param radiance: The luminosity profile (ergs/s), must include l-dependence 
 58      @type radiance: Radiance()

 61                  none. 
 62                   
 63                  (default: None) 
 64      @type T: Temperature() 
 65      @keyword P: The porosity of the grain (or equivalent, to represent  
 66                  effective grain surface). Default is the spherical case. Is 
 67                  given as the ratio of vacuum per unit volume of a grain. 
 68                   
 69                  (default: 0) 
 70      @type P: float 
 71      @keyword alpha: Sticking coefficient (Kwok 1975, Van Marle 2011). Default is 
 72                      current value in MCP/GASTRoNOoM. Should be 0.25 according to 
 73                      Kwok and van Marle. 1-alpha gives the fraction of elastic 
 74                      collisions. 
 75       
 76                      (default: 0.) 
 77      @type alpha: float 
 78      @keyword mu: The mean molecular weight. Default is a wind with purely H2.  
 79                   To get the GASTRoNOoM value of 1.4, set fH to 1.5. 
 80       
 81                   (default: 2.0) 
 82      @type mu: float 
 83      @keyword w_thermal: Type of thermal velocity.  
 84                             - sqrt(2kT/m): most probably speed df(v)/dr = 0 
 85                             - sqrt(8/pi kT/m): mean speed (f(v) * v dv)/f(v) dv 
 86                             - sqrt(3kT/m): rms speed sqrt((f(v) * v2 dv)/f(v) dv) 
 87                             - Kwok version: 3/4 sqrt(3kT/m) 
 88                             - Epstein: 4/3 sqrt(8kT/mpi) 
 89                          Choose from: epstein, kwok, prob, mean, rms, none  
 90                            
 91                          (default: none) 
 92      @type w_thermal: str 
 93       
 94      @return: The drift velocity (cm/s) as a function of r and a 
 95      @rtype: array/float 
 96       
 97      ''' 
 98       
 99      #-- Define some constants  
100      c = cst.c.cgs.value 
101      m_p = cst.m_p.cgs.value 
102      k_b = cst.k_B.cgs.value 
103      w_thermal = w_thermal.lower() 
104       
105      #-- Check whether v and mdot are constants 
106      vi = v.eval(r) if isinstance(v,Velocity) else v 
107      mdoti = mdot.eval(r) if isinstance(mdot,Mdot.Mdot) else mdot 
108      mdoti = (mdoti*u.Msun/u.yr).to(u.g/u.s).value 
109       
110      #-- Integrate the opacity and luminosity profiles over wavelength 
111      #   For this: calculate the emitting surface of the central source 
112      L = radiance.getLuminosity(l=l,ftype='flambda') 
113      OpacL = trapz(x=l,y=opac.eval(l)*L) 
114       
115      #-- Calculate the drift for each grain size 
116      #   1) create the 2d array with r-dependent and a-dependent 1d arrays 
117      #   2) Then add in anything that's constant 
118      #   Note that Q(a) = kappa*4/3*a*sd*(1-P)^(2/3) 
119      arr = np.outer(vi/mdoti,a) 
120      vK = np.sqrt(arr/(c*(1-alpha))*OpacL*4./3.*sd*(1.-P)**(2./3.)) 
121       
122      #-- Calculate the thermal term (see Decin 2006) or return vK (default) 
123      if not w_thermal in ['kwok','mean','rms','prob','epstein']:  
124          return vK 
125   
126      #-- RMS, ie sqrt(Int(f(v) v^2 dv)/Int(f(v) dv)) 
127      if w_thermal == 'rms': 
128          vT = (3.*k_b*T.eval(r)/(mu*m_p))**0.5 
129           
130      #-- Most probable speed: d(f(v)/dv = 0) 
131      elif w_thermal == 'prob': 
132          vT = (2.*k_b*T.eval(r)/(mu*m_p))**0.5 
133           
134      #-- Mean, ie Int(f(v) v dv)/Int(f(v) dv) ==> likely the most appropriate 
135      elif w_thermal == 'mean': 
136          vT = (8.*k_b*T.eval(r)/(mu*m_p*np.pi))**0.5 
137       
138      #-- Implementation by Epstein 1924, with the correct factor 4./3. 
139      elif w_thermal == 'epstein':  
140          vT = 4/3.*(8.*k_b*T.eval(r)/(mu*m_p*np.pi))**0.5 
141           
142      #-- kwok: Unknown factor 3/4 and rms velocity 
143      else:  
144          vT = 0.75*(3.*k_b*T.eval(r)/(mu*m_p))**0.5 
145       
146      #--  Make sure the right axis of vT is multiplied with vK^-1 
147      vT = np.transpose([vT]) 
148      xT = 0.5*(vT/vK)**2. 
149      factor = ((1.+xT**2)**0.5-xT)**0.5 
150       
151      return factor*vK
